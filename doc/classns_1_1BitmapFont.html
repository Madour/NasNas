Creates a BitmapFont from a Texture.

This constructor will use the default character map and advance map. Make sure your characters in the texture are placed in the following order (first character is a space): !"#$%&'()*+,-./0123456789:;<=>?[\]^_`abcdefghijklmnopqrstuvwxyz{|}~
